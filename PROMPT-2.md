# Design Patterns — Premium Flat Icon Prompt Blueprint

This document contains a comprehensive set of highly specific, minimalist, and vector-optimized AI image generation prompts to produce premium **1:1 flat design icon logos** for every page on the platform.

These prompts are engineered to generate high-contrast, simple, flat-colored graphical symbols with clean negative space. They are optimized for direct conversion to vector SVGs in design tools like Figma (using Image Trace or SVG vectorization plugins).

---

## 🎨 Global Icon Design System Recipe

To ensure all 45 icons belong to the same visual family while conveying highly distinct software design concepts, every prompt adheres to a strict flat design recipe.

### The Icon Blueprint Rules
* **Aesthetic**: Minimalist flat 2D graphic design, solid vector logo icon, high-contrast, sharp geometric contours.
* **Palette Depth**: Strict maximum of **2 to 3 solid flat colors** (Deep Indigo, Electric Cyan/Teal, Soft Coral/Orange, and pristine White). **Zero gradients, zero drop shadows, zero 3D depth/textures, and no realistic rendering details.**
* **Layout**: Clean, solid, flat background (sleek solid charcoal-black background `#121214`) to allow perfect contrast and easy automated vector tracing.
* **SVG Convertibility**: Bold strokes, clear negative space, unified solid fills, and no complex shading or tiny details.
* **Output Aspect Ratio**: Strict 1:1 square ratio (`--ar 1:1`).
