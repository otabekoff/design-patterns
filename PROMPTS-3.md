# Design Patterns — Hand-Drawn Doodle Illustrative Prompt Blueprint

This document contains a comprehensive set of highly specific, playful, and educational AI image generation prompts to produce custom, high-density **1:1 hand-drawn doodle illustrations** matching the classic **Refactoring Guru** aesthetic.

These prompts use personified characters, funny software analogies, and watercolor textures, optimized for generating clean diagrams that can be easily traced as vector SVGs.

---

## 🎨 Global Doodle Style Framework

To achieve the exact playful, sketchy look of hand-drawn textbooks, all prompts in this blueprint adhere to the following style framework.

### The Refactoring Guru Design Recipe
* **Aesthetic**: Hand-drawn educational vector line-art illustration, playful cartoon sketch, soft watercolor washes, slightly imperfect pencil doodles.
* **Line Art**: Organic, sketchy charcoal-dark outlines, expressive hand-drawn contours. No artificial vector gradients or glowing 3D renders.
* **Color Palette**: Minimalist pastel watercolors — gentle sky blue/teal, bright yellow-orange, warm cream/tan, and soft mint green.
* **Background**: Solid clean white background (`#ffffff`), fully isolated with zero shadows or background clutter for easy Figma tracing.
* **Composition**: Flat 2D isometric or front-facing perspective, centered object framing, 1:1 aspect ratio (`--ar 1:1`).
